O que é CSS?

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata, em português), é uma linguagem de estilo usada para descrever a apresentação de documentos escritos em HTML ou XML. O CSS permite que os desenvolvedores separem a estrutura e o conteúdo do design e da formatação de um site. Com o CSS, é possível controlar aspectos como cores, fontes, espaçamentos e layouts de uma página web. Ele oferece um modo eficiente e flexível para aplicar estilos a vários elementos simultaneamente e ajustar a aparência de uma página sem alterar o seu conteúdo HTML. O CSS pode ser usado para estilizar um documento muito básico de texto — por exemplo, alterando a cor e tamanho dos títulos e links. Pode ser usado para criar layout — por exemplo, transformando uma simples coluna de texto em um layout com uma área de conteúdo principal e um sidebar (barra lateral) para as informações relacionadas. Pode até ser usado para efeitos tais como animação. Dê uma olhada nos links deste parágrafo, para ver exemplos específicos.
